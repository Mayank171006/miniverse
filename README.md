# MiniVerse 🎮

**Play. Compete. Achieve.**

MiniVerse is a browser-based gaming platform featuring multiple mini-games, achievements, leaderboards, and player profiles powered by Firebase.

🌐 **Live Demo:** https://mayank171006.github.io/miniverse/

---

## ✨ Features

### 🎮 Games

MiniVerse currently includes four games:

- 🐍 **Snake** – Eat, grow, and survive while chasing the highest score.
- ⚡ **Reaction Test** – Measure your reflexes and test your reaction speed.
- 🧠 **Simon Says** – Memorize and repeat increasingly difficult sequences.
- 🧩 **2048** – Merge tiles strategically to reach higher values and maximize your score.

---

### 🏆 Achievement System

Unlock achievements by reaching milestones across different games.

Achievements are categorized and automatically unlocked based on your gameplay statistics.

---

### 📊 Leaderboards

Compete with other players and compare your performance.

Features include:

- Game-specific leaderboards
- Top player rankings
- Personal ranking display
- Firebase-powered score tracking

---

### 👤 Profile Dashboard

Track your progress across all games.

The profile page displays:

- Best scores
- Average scores
- Games played
- Personal statistics for each game

---

### 🔐 Authentication

Google Sign-In integration powered by Firebase Authentication.

Signing in allows players to:

- Save scores
- Track achievements
- Appear on leaderboards
- Maintain a personal profile

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/Mayank171006/miniverse.git
```

Move into the project directory:

```bash
cd miniverse
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
